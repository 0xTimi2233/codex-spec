# Roadmap

PM owns this file. The main thread uses it to pick the next ready milestone.

Status values:

- `planned`: known but not ready.
- `ready`: dependencies are satisfied and auto mode may start it.
- `active`: currently being executed.
- `blocked`: waiting for a decision or external condition.
- `done`: finished and verified.

| ID | Milestone | Status | Depends on | Exit criteria | Blocker |
|---|---|---|---|---|---|
| M0 | Initialize workflow | ready | - | AGENTS.md, .codex, .agents, agentflow, and .agentflow exist | - |
